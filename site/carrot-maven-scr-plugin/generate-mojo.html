carrotscr:generate

Full name:

com.carrotgarden.maven:carrot-maven-scr-plugin:2.0.1-SNAPSHOT:generate

Description:

Generate component descriptors form annotated java classes.

Attributes:

  • Requires a Maven project to be executed.
  • Requires dependency resolution of artifacts in scope: test.
  • Binds by default to the lifecycle phase: process-classes.

Required Parameters

Name Type Since Description
excludeFileNameRegex String - Exclude java *.class files with this file name regex expression.
Default value is: .*-.*.
excludedServices Set - Collection of names of unwanted component service interfaces.
Default value is: java.lang.Cloneable.
isIncludeEmptyDescriptor boolean - Should include an empty component descriptor?
Default value is: true.
isIncludeGeneratedDescritors boolean - should generated descriptor resource files be included in final bundle?
Default value is: true.
isProcessMainClasses boolean - Should "main" classes be processed?
Default value is: true.
isProcessTestClasses boolean - Should "test" classes be processed?
Default value is: false.
outputExtensionSCR String - Default extension used for generated DS component descriptor files.
Default value is: xml.
outputMainClasses File - Location of compiled "main" class files.
Default value is: ${project.build.outputDirectory}.
outputTestClasses File - Location of compiled "test" class files.
Default value is: ${project.build.testOutputDirectory}.
properPackaging Set - Collection of names of known maven project packaging types for which to invoke this plug-in.
Default value is: bundle.
targetDirectorySCR String - Location of generated DS component descriptor files in final bundle.
Default value is: OSGI-INF/service-component.

Optional Parameters

Name Type Since Description
eclipseSettings Map - Map of "key=value" settings for Eclipse m2e-scr connector plugin.
